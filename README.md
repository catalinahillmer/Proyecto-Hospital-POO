 Sistema de Gestión Hospitalaria - POO

Este proyecto es una aplicación de escritorio desarrollada en Java que gestiona el personal de un centro médico.

Criterios de Evaluación Implementados
* **Interfaz:** Uso de la interfaz `Trabajador` con métodos abstractos.
* **Herencia:** Superclase `Personal` y tres subclases: `Medico`, `Enfermero` y `Administrativo`.
* **Polimorfismo:** Uso de `instanceof` para identificar dinámicamente los roles en una colección.
* **GUI:** Interfaz gráfica completa creada con Swing para interactuar con las entidades.

Cómo Ejecutar
1. Clonar el repositorio.
2. Abrir en cualquier IDE compatible con Java (NetBeans, IntelliJ, Eclipse).
3. Ejecutar la clase `HospitalGUI.java`.
